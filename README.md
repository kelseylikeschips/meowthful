# meowthful
Repository name: meowthful\
Project title: meowthful\
Project description: An application designed to track a cat's meals, to ensure that they are not fed multiple times for one meal allocation by allowing users within a household to notify when they have fed the family cat(s) for the specified meal time. It can also be used to track medication intake.
